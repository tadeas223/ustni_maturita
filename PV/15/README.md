# Programovací jazyky - vlastnosti, srovnání, popis způsobu tvorby i běhu programů

## Funkční / Procedurální / OOP

**Funkční**
- Žádný globální stav
- Hlavní jsou funkce
- Všechna data jsou konstantní
- Haskell, Lisp

**Procedurální**
- Struktury obsahují pouze data
- Není dědičnost
- C, Rust

**OOP**
- Třídy
- Dědění vlastností
- Java, C#, Python

# Low-Level / High-Level

Low-Level 
- Většinou rychlejší
- Méně abstrakcí
- Většinou kompilované
- C, C++, Rust

High-Level
- Více abstrakcí
- GC
- Java, C#, Python
