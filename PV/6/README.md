# Asymptotické paměťové a časové složitosti

Způsob jak počítat náročnost algoritmů
- O(n) -> Lineární složitost
- O(1) -> Konstantn složitost

Nejefektivnější algoritmy mají logaritmickou, mebo lineární složitost
Nejpomalejší algoritmy mají exponenciální a nebo kvadratickou složitost
